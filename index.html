Mobile Behavior

Nav items scroll horizontally on mobile with snap-to-item behavior. No hamburger menu needed.

Status Bar Color (iOS/Android)

Match the browser status bar to your navbar color using the theme-color meta tag:

<!-- Light navbar -->
<meta name="theme-color" content="#ffffff">

<!-- Dark navbar -->
<meta name="theme-color" content="#1e1b4b">

Hide Top Row on Scroll (JS)

On mobile, hide the logo/buttons row when scrolling down, reveal when scrolling up:

<script src="js/framework.js"></script>

const navbar = document.querySelector('.navbar');
initNavbarScroll(navbar);

// Optional: custom threshold (default: 10px)
initNavbarScroll(navbar, { threshold: 20 });

Smooth fade-out

Removing the gradient abruptly looks jarring. Use easeOutGradient() for a polished transition.

<script src="js/framework.js"></script>

// Fade out smoothly
easeOutGradient(element);

// Need to run code after? It's a Promise:
await easeOutGradient(element);
doSomethingNext();
